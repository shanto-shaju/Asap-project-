ABOUT WI-FI MANAGER

Wi-Fi Manager is a software that helps manage Wi-Fi networks. It provides scanning of nearby networks, registration of Wi-Fi details, and monitoring of connected devices for security and management.

Key features include:

  • Scan available Wi-Fi networks
  • Register and store Wi-Fi details securely
  • Monitor connected devices
  • Simulated security features for demonstration

This project is developed using Python Flask and Bootstrap for a clean, responsive user interface.
